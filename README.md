# Data Analytics Portfolio 📊

This repository is my personal portfolio for learning and practicing **data analytics, Excel, SQL and Python**.

The goal is to collect small but clear projects that show practical skills: data cleaning, analysis, reporting, dashboards, SQL queries and simple automation.

---

## 🧭 Repository structure

```text
data-analytics-portfolio/
├── excel/          # Excel dashboards, reports and spreadsheet projects
├── sql/            # SQL practice tasks and case studies
├── python/         # Python mini-projects and notebooks
├── projects/       # Complete end-to-end analytics projects
├── data/           # Public sample datasets only
├── images/         # Screenshots, charts and dashboard previews
├── requirements.txt
└── README.md
```

## 🎯 Learning roadmap

### 1. Excel
- formulas and functions;
- pivot tables;
- filtering and sorting;
- charts and dashboards;
- business reports.

### 2. SQL
- SELECT, WHERE, ORDER BY;
- JOINs;
- GROUP BY and aggregations;
- subqueries;
- analytical case studies.

### 3. Python
- basics of Python;
- pandas;
- reading and cleaning data;
- basic visualizations;
- small automation scripts.

### 4. Portfolio projects
Each project should include:

- a short description;
- the business question;
- data source;
- tools used;
- key steps;
- screenshots or charts;
- final conclusions.
