# Excel Projects 📗

This folder is for Excel-based analytics projects.

## What to upload here

- dashboards;
- reports;
- pivot table practice;
- screenshots of dashboards;
- short project descriptions.

## Good project structure

For each Excel project, create a separate folder:

```text
excel/
└── sales-dashboard/
    ├── README.md
    ├── dashboard.xlsx
    └── images/
        └── dashboard-preview.png
```

## README checklist for each project

- What is the goal of the dashboard?
- What data is used?
- What metrics are shown?
- What insights were found?
- What can be improved later?
