# Python Mini-Projects 🐍

This folder is for Python learning projects, notebooks and simple automation scripts.

## What to practice

- Python basics;
- pandas;
- reading CSV / Excel files;
- data cleaning;
- basic charts;
- small automation scripts.

## Suggested structure

```text
python/
└── data-cleaning-mini-project/
    ├── README.md
    ├── notebook.ipynb
    ├── script.py
    └── output/
```

## Project README checklist

- What problem does the script or notebook solve?
- What data is used?
- What libraries are used?
- What steps are completed?
- What result was received?

## Rule

Keep projects simple, clear and well explained. A small finished project is better than a large unfinished one.
